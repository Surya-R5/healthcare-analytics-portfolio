### 2. Breast Cancer Screening Disparities Dashboard
Tools: Tableau · CDC BRFSS Data  
Description: Interactive dashboard analyzing breast cancer screening disparities across 50 US states by demographic groups. Visualizes gaps in screening rates by race/ethnicity, age group, and state to identify priority intervention areas.

Key Features:
- State Screening Performance: Bar chart ranking states by screening rates
- Disparities by Demographics: Race/ethnicity breakdown by age groups
- Disparities Heatmap: State × Race/Ethnicity grid showing gaps visually (red = bigger gap, green = close to target)

Key Skills Demonstrated:
- Tableau dashboard design and interactivity
- Data storytelling: turning data into actionable insights
- Healthcare equity analysis
- Identifying vulnerable populations

**Live Dashboard**: [View Interactive Dashboard](https://public.tableau.com/views/BreastCancerScreeningDisparities/BreastCancerScreeningDisparitiesDashboard?:language=en-US&publish=yes&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)
